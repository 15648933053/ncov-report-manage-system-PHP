标签（tag）说明：每一个成员可以被设置不同的标签（tag）
全局标签：由机构管理员创建并分配权限,该标签可以给白名单内的所有人员打标签
部门标签：由部门管理员创建并分配权限,该标签只能给白名单内的所属部门人员打标签
标签管理员：通常为机构内的:跨部门的组长,项目组组长、学校辅导员、班主任、班长等
标签的作用: 主要方便标签管理员查看填报信息,以便协助催促上报,可在手机端方便查看信息
标签内的成员名单管理：由部门管理员通过Excel批量设置
